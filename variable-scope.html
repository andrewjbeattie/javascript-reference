Variable Scope

The parameters of a function are only accessible inside that function.
That means from the opening { until the corresponding closing }.

For example:

// number here is NOT defined

function double(number) {
    // number can only be used INSIDE this function
    return number * 2
}

// number here is NOT defined

This concept is called the scope of a variable.

You must know that the parameter is only accessible inside the function, and nowhere else.

Storing the result of a function

Since functions return a result, you can in fact store that result in a variable. Here's an example:

Here we define the function that will double the parameter "number"
which is passed to it through an argument stored in a variable:



function double(number) {
console.log(number * 2);
return number * 2
}

And here we store the variables with each argument:


let firstResult = double(4);
let secondResult = double(10);



Notice how we save the result of double(4) into the variable firstResult.
The variable result now contains the result of the double(4) function call, which is 8.
So the value of firstResult is now 8.
Similarly, the secondResult variable has the result of double(10) which is 20.

function sum(a, b){
console.log(a + b);
return a + b
}
